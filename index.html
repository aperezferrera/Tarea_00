Explicación de Tarea 0.0

Esta tarea consiste en crear un archivo HTML básico que contenga una estructura mínima de una página web con un título y un párrafo explicativo.

En este HTML vamos a hablar sobre la página web de Bono Sevilla. Que te permite comprar cinco bonos por DNI.

Basicamente en la página te dice que puedes comprar 5 bonos por cada DNI que se introduzca.

El bono cuesta 15€ y tiene un valor de 25€ para gastar en las distintas tiendas afiliadas

El entorno cliente literalmente es la parte donde tenemos que introducir nuestros datos personales y tarjeta de débito

El entorno servidor es el que se encarga de:
- Ponerte en una cola virtual
- Validar los datos introducidos
- Comprobar que el DNI no haya comprado ya 5 bonos
- Procesar el pago
- Enviar un correo con el bono en formato PDF
- Guardar los datos en una base de datos
