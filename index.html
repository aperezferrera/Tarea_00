Explicación de Tarea 0.0

Esta tarea consiste en crear un archivo HTML básico que contenga una estructura mínima de una página web con un título y un párrafo explicativo.

En este HTML vamos a hablar sobre la página web de Bono Sevilla. Que te permite comprar cinco bonos por DNI.

Basicamente en la página te dice que puedes comprar 5 bonos por cada DNI que se introduzca.

El bono cuesta 15€ y tiene un valor de 25€ para gastar en las distintas tiendas afiliadas

El entorno cliente literalmente es la parte donde tenemos que introducir nuestros datos personales y tarjeta de débito

El entorno servidor es el que se encarga de:
- Tener disponible un input type text para el DNI
- Tener disponible un input type text para el correo electrónico
- Tener disponible un input type number para la tarjeta de débito
- Tener disponible un botón submit para comprar el bono
- Ponerte en una cola virtual con un arrayList que cuente cuanto te falta para entrar y en qué posición de la cola
- Validar los datos introducidos en los campos y que se respeten los máximos y mínimos establecidos en el código
- Comprobar que el DNI no haya comprado ya 5 bonos con un condicional. Si intentas comprar más de 5 bonos te salta un mensaje diciendo: "Solo 5 bonos por DNI"
- Procesar el pago
- Enviar un correo con el bono en formato PDF. Al correo electrónico que hemos introducido antes.
- Guardar los datos en una base de datos. Al guardarlo dentro de una lista o arrayList ya quedas registrado y no puedes volver a hacerlo hasta la próxima vez.
